type: project
---

User wants persistent agent memory across sessions and intelligent model/agent switching/creation. These are core features for daily use.

**Why:** The agent should learn from past conversations and adapt. Multiple models/agents for different tasks (coding, research, writing) without manual config editing.

**How to apply:** Prioritize memory and agent profiles in future Towel work. These are the features that make it a personal assistant vs. a chat wrapper.
