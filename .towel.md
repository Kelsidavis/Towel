# Towel — Don't Panic.

Local AI assistant powered by MLX on Apple Silicon.

## Stack
- Python 3.11+, MLX/mlx-lm for inference
- WebSocket (websockets) + HTTP (Starlette/Uvicorn) gateway
- Rich for CLI, Pydantic for config, Click for commands
- JSON file persistence for conversations and memories

## Structure
- src/towel/agent/ — MLX runtime, streaming, tool parsing, context window, project context
- src/towel/gateway/ — WebSocket + HTTP server, session management
- src/towel/skills/ — Skill registry, 4 builtins (filesystem, shell, web, memory), auto-loader
- src/towel/memory/ — Persistent agent memory across sessions
- src/towel/persistence/ — Conversation store, search, export
- src/towel/cli/ — CLI commands + chat slash commands
- src/towel/web/ — Single-page chat UI with sidebar and session persistence

## Conventions
- All async, event-driven architecture
- Config in TOML (~/.towel/config.toml)
- Agent profiles for model/persona switching
- Hitchhiker's Guide to the Galaxy theming throughout
